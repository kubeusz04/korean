📖 Introduction

  • Full explanation

In Korean, words like I and You can have extra particles attached to them. These particles help show what the sentence is talking about.

One of the most common particles is:

는

It is called the topic marker.

The main structure for compliments:

너는+description

너는 = You (Ty / jeśli chodzi o ciebie)

💬 Examples

You are the best. 너는 최고야 Jesteś najlepsza / najlepszy.
You are amazing. 너는 대단해 Jesteś niesamowita / niesamowity.
You are beautiful. 너는 아름다워 Jesteś piękna / piękny.
Amazing Kuba 대단한 쿠바 Niesamowity Kuba. — adjective before the name (different pattern!)

Remember

Compliment pattern

  • 너는 + description
  • 너는 최고야
  • 너는 대단해

Name pattern

  • adjective + name
  • 대단한 쿠바
